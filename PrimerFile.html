PrimerFile

A PrimerFile is a file that contains a list of primers. Each primer is represented as a line in the file, and the fields are separated by tabs. The fields are:

  • Name: The name of the primer.
  • Sequence: The sequence of the primer.
  • Length: The length of the primer.
  • GC Content: The GC content of the primer.
  • Melting Temperature: The melting temperature of the primer.
